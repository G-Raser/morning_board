# Changelog

## v1.1.1

- 修复刷新页面后今天已完成列表不显示的问题。
- 修复复制弹窗元素尚未解析时绑定事件导致的前端初始化中断。
- 不涉及数据库结构变化，只需覆盖 `static/index.html`。

## v1.1.0

- 修复新完成记录的时间显示为 UTC 的问题。
- 新完成记录改为用电脑本地时间写入 `created_at`。
- DDL 新增 `due_time` 字段，支持具体到小时和分钟。
- DDL 列表显示完整截止时间。
- DDL 倒计时支持小时/分钟级别。
- 兼容旧数据库：旧任务没有时间时仍然按日期截止显示。

## v1.0.0

当前稳定可用版候选。

包含：

- SQLite 本地持久化
- 上板 / 下板双区域 UI
- 昨天写给今天看
- 今天写给明天看
- DDL 未完成保留
- DDL 标记完成后进入历史
- 按日期查看完成记录
- 亮色 / 暗色切换
- 暗色模式启动闪烁修复
- 每日提醒设置
- 复制今日存档
- 剪贴板权限失败时弹出手动复制框
- 支持通过 Tailscale 手机访问
